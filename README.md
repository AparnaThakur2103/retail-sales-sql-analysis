# Retail Sales Analysis using SQL

## Overview
This project demonstrates end-to-end SQL-based business analysis on a retail e-commerce dataset. The analysis focuses on revenue computation, customer behavior, product performance, and operational metrics.

## Database Structure
The dataset consists of four relational tables:
- customers
- products
- orders
- order_items

These tables are connected via primary and foreign key relationships to support transactional analysis.

## Key Analysis Areas
- Revenue and profitability analysis
- Monthly revenue growth
- Customer spending behavior
- RFM segmentation
- Product category performance
- Cancellation and return rate
- Payment mode performance

## SQL Concepts Used
- Joins
- Subqueries
- Aggregations & GROUP BY
- Window Functions
- CTEs
- CASE statements
- Views
- Date & String functions
